# V1.0 Requirements Outline

This document outlines the requiremnts for the V1.0 UAV. 

## Requirements Definition

1. V1.0 shall execute takeoff unassisted and sustain flight without payload.
2. V1.0 shall respond to helm control commands prompted by a remote operator.
3. V1.0 shall execute up/down/left/right/forward/backward commands.
4. V1.0 shall execute helm control commands individually and sequentially.
5. V1.0 shall execute an emergency landing procedure in the event of a critical fault.
6. V1.0 shall return to takeoff location whem prompted.
