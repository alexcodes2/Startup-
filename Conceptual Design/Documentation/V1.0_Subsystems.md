## List of Subsystems

1. Mechanical System
   - Material selection and procurement
   - CAD Design of frame and first model
 
2. Power System 
   - BMS 
   - Surge protection
   - MCU selection
   - Electronic Speed Controllers
     
3. Communication System
   - Bidirectional 
   - Wireless communication
   - Executes instructions
   - Obstacle avoidance sensor (LiDAR)

4. Flight Management System
   - Motor selection
   - PID controller
   - Time of flight sensor (1D LiDAR)  
   - GPS data
   - IMU sensor

5. Software Architecture
   - Finite State Machine
   - GUI
   - I/O control
